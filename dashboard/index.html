Counterfactual Regret Minimization

CFR is a self-play algorithm that iteratively improves strategies by minimizing counterfactual regret at each information set. The average strategy over all iterations converges to a Nash equilibrium.

Counterfactual Value
vi(σ, I) = Σh∈I π-iσ(h) · Σz∈Zh πσ(h,z) · ui(z)
Regret Matching
σT+1(I, a) = RT+(I, a) / Σb RT+(I, b)

🎯 Vanilla CFR

Full game tree traversal each iteration. Computes exact counterfactual values for all information sets simultaneously.

O(|H|) per iteration

⚡ CFR+

Three key modifications: non-negative regrets (floor at 0), alternating player updates, and linear averaging (weight by iteration number).

Faster constant

🎲 External Sampling MCCFR

Sample opponent + chance actions, traverse all of traversing player's actions. Unbiased regret estimates with lower variance than outcome sampling.

O(|A|) per iteration

📊 DCFR

Discounts early iterations via hyperparameters α, β, γ. Positive regrets discounted by tα/(tα+1), negative by tβ/(tβ+1).

Tunable convergence
